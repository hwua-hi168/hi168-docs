# 教育

Hi168 平台提供完整的在线教育功能，包括课程学习、直播授课、课程制作、教务管理、求职就业和班级管理。

---

## 功能模块

| 模块 | 说明 |
| --- | --- |
| **课程学习** | 浏览课程列表，加入课程，在线观看视频 |
| **直播** | 参加实时直播课程，支持互动和回放 |
| **课程制作** | 创建、编辑和发布课程内容 |
| **教务管理** | 管理班级和成员名单，分配课程与任务 |
| **求职就业** | 浏览和发布岗位信息 |
| **我的班级** | 查看已加入的班级和班级详情 |

请从左侧导航栏选择具体模块查看详细操作说明。
