# 在线学习

## 概述

Hi168 平台的 **在线学习** 模块为学习者提供了完整的在线教育体验，涵盖课程浏览、加入学习、视频播放、实验操作、直播互动等核心功能。平台拥有 60+ 门课程，覆盖人工智能、数据库、云计算与大数据、区块链等热门技术领域，按难度分为入门、初级、中级、高级四个等级，另设框架和案例两个专项类型。

---

## 一、课程学习

### 1.1 课程列表

进入「课程学习」模块，系统展示所有已发布的课程列表。用户可通过以下条件进行筛选与检索：

- **课程类型**：免费课程、付费课程
- **课程分类**：人工智能、数据库、云计算与大数据、区块链等
- **难度等级**：入门、初级、中级、高级、框架、案例
- **关键字搜索**：在搜索框中输入课程名称或关键词进行模糊检索

### 1.2 加入课程

课程分为免费和付费两种类型，加入方式有所不同：

| 类型 | 加入方式 |
| --- | --- |
| **免费课程** | 直接点击「加入课程」即可加入，无需额外操作 |
| **付费课程** | 需先完成购买流程，支付成功后方可加入学习 |

### 1.3 课程详情

点击任一课程可进入课程详情页，查看该课程的完整信息。以下为详情页包含的主要内容板块：

| 板块 | 说明 |
| --- | --- |
| **课程介绍** | 课程的整体概述、教学目标、适用人群等基础信息 |
| **课程目录** | 按章节组织的课程大纲结构，展示所有教学环节 |
| **直播日历** | 与该课程关联的直播教学安排与时间表 |
| **课程附件** | 课程配套的补充资料与下载资源 |
| **课程评论** | 学习者对课程的评价与反馈 |

### 1.4 课程播放

点击课程章节中的视频、练习或课件图标，可进入课程播放页面。播放页面集成了多种教学内容与交互功能：

#### 1.4.1 核心教学功能

| 功能 | 说明 |
| --- | --- |
| **视频播放** | 在线播放课程教学视频，支持播放进度控制 |
| **实验手册** | 展示当前章节配套的实验操作指南 |
| **课堂练习** | 章节随堂练习题，帮助巩固学习成果 |
| **课件预览** | 在线查看课程演示文稿 |
| **课堂笔记** | 支持在学习过程中记录个人笔记 |

#### 1.4.2 辅助学习功能

| 功能 | 说明 |
| --- | --- |
| **分屏** | 将学习界面分为多个区域，可同时查看视频和实验环境 |
| **创建实验环境** | 在课程播放页面中一键部署与当前章节关联的实验环境，即可在学习理论的同时进行实操练习 |

> **亮点**：Hi168 的课程播放页面将理论学习与实践操作深度融合，学习者可在同一界面内观看教学视频、阅读实验手册、完成练习并操作实验环境，实现"学练一体"。

## 二、直播教学

### 2.1 功能说明

直播模块为教育工作者提供实时在线教学能力。主要特性包括：

- **高可靠性**：支持多种设备接入，保障学生访问的稳定性。
- **多种交互方式**：支持语音、文字、图片、文件等多形式互动，师生可实时沟通。
- **灵活性**：支持班级直播、分组直播等多种形式，支持多用户同时观看。

### 2.2 直播列表

进入「直播」模块后，系统展示当前所有直播课程，状态分为三类：

| 状态 | 说明 |
| --- | --- |
| **正在直播** | 直播正在进行中，用户可直接进入观看 |
| **未直播** | 直播尚未开始，用户可提前加入等待 |
| **已结束** | 直播已结束，可查看回放（如已开启） |

### 2.3 加入直播

对于状态为「未直播」的课程，用户可提前点击「加入直播」进行预约。当直播正式开始后，即可进入直播间参与学习。

### 2.4 直播回放

直播结束后，平台将自动录制直播过程并生成视频文件上传至云端。教师可将录制视频拉取下来，供学生进行直播回放观看。

### 2.5 创建直播（教师端）

教师可通过以下流程创建直播课程：

1. 进入直播模块，点击「创建直播」。
2. 填写直播信息（直播标题、时间、关联班级等）。
3. 确认创建。
4. 到达直播时间后，使用 OBS 等推流软件进行直播。

### 2.6 编辑直播

教师可对已创建的直播课程进行编辑，修改直播标题、时间、描述等信息。

### 2.7 删除直播

对于确定不再需要的直播课程，教师可将其从列表中删除。

### 2.8 直播配置（OBS 推流）

平台直播采用 OBS 软件进行推流。在「直播配置」中可查看推流所需的配置信息（如推流地址、推流密钥等），将其复制到 OBS 软件中即可开始推流。

> 详细的 OBS 直播使用方法，请参阅平台内的「操作手册」。

### 2.9 移动端观看

平台支持移动端观看直播。用户可扫描微信小程序二维码，通过手机端进入直播间观看教学内容。

## 三、我的班级（学生视角）

### 3.1 功能说明

「我的班级」模块面向学生用户，用于管理和参与所属班级的各项教学活动。

### 3.2 班级列表

进入「我的班级」后，展示学生当前加入的所有班级列表。

### 3.3 班级详情

点击进入某个班级，可查看以下详细信息：

- **班级基本信息**：班级名称、授课教师等。
- **培训课程**：该班级关联的课程内容。
- **直播安排**：班级内的直播教学计划。
- **编辑操作**：可点击右上角「编辑」修改班级相关信息（如有权限）。

## 四、学习路径建议

1. **浏览课程中心**：根据自身兴趣和学习目标，在课程列表中筛选适合的课程。
2. **加入课程学习**：加入课程后，按照章节顺序依次学习视频、完成练习、操作实验。
3. **参与直播互动**：关注直播日历，准时参加直播课程并积极互动。
4. **回顾与巩固**：利用直播回放、课堂笔记和存储桶中的课程资料进行复习。
5. **实验实践**：在课程播放页面中创建实验环境，将理论知识应用到实际操作中。
