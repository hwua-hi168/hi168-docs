# 数据集及模型

## 概述

Hi168 平台的 **数据集及模型** 模块为人工智能教学与研究提供数据资源与模型管理能力。该模块包含 **AI 模型** 和 **数据集** 两大子功能，支持用户浏览、管理和使用各类机器学习模型与训练数据集，为 AI 实验与项目开发提供基础数据支撑。

---

## 一、AI 模型

### 1.1 功能说明

AI 模型管理提供对预训练模型及用户自定义模型的统一管理能力。用户可在此浏览平台提供的公共模型资源，也可上传和管理自己训练的模型文件。

### 1.2 核心功能

| 功能 | 说明 |
| --- | --- |
| **模型浏览** | 查看平台提供的公共 AI 模型列表，按类型、领域等维度进行筛选 |
| **模型详情** | 查看模型的版本信息、参数规模、适用场景、性能指标等元数据 |
| **模型管理** | 对个人上传或训练的模型进行版本管理、描述编辑、删除等操作 |
| **模型调用** | 在实验环境或 Notebook 中加载并使用模型，进行推理或继续训练 |

### 1.3 典型使用场景

- 在深度学习课程中加载预训练模型进行迁移学习实验。
- 上传自行训练的模型供团队成员共享和评估。
- 浏览平台模型库，选择合适的基础模型作为项目起点。

---

## 二、数据集

### 2.1 功能说明

数据集管理为用户提供结构化的数据资源管理服务。用户可浏览平台公共数据集，也可上传和维护自有数据集，用于模型训练、实验验证和课程教学。

### 2.2 核心功能

| 功能 | 说明 |
| --- | --- |
| **数据集浏览** | 查看平台公共数据集列表，按领域、数据类型、规模等维度筛选 |
| **数据集详情** | 查看数据集的描述信息、数据规模、字段说明、样本预览等 |
| **数据集上传** | 将本地数据文件上传至平台，创建自有数据集 |
| **数据集管理** | 对自有数据集进行版本更新、描述编辑、权限设置、删除等操作 |
| **数据集引用** | 在 Notebook 或应用环境中直接引用数据集，免去手动下载和配置 |

### 2.3 典型使用场景

- 教师上传课程配套的实验数据集，学生可在实验环境中直接引用。
- 在模型训练过程中引用平台数据集，简化数据准备流程。
- 团队协作时共享标注数据集，提升协作效率。

---

## 三、最佳实践

1. **规范化元数据**：上传数据集或模型时，完整填写描述信息、版本号、适用场景等元数据，方便后续检索和复用。
2. **版本管理**：对数据集和模型进行版本化管理，确保实验可复现。
3. **合理设置权限**：根据需求将数据集和模型设为公开或私有，保护敏感数据的同时促进资源共享。
4. **配合 Notebook 使用**：在 Notebook 中直接加载数据集和模型，实现数据探索、模型训练、结果可视化的一体化工作流。
