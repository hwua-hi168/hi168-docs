# 平台与个人设置

## 概述

Hi168 平台的 **桌面中心** 是整个平台的控制中枢，集成了所有功能模块的入口和系统级配置。用户的日常操作均从桌面中心发起。本章节介绍桌面中心的核心功能，包括个人信息管理、费用中心、桌面设置、云端互联以及帮助系统。

---

## 一、桌面中心

### 1.1 功能说明

云桌面是 Hi168 平台的统一操作入口，采用可视化桌面界面风格，将所有功能模块以直观的图标方式呈现。用户可通过桌面中心快速访问应用中心、课程学习、直播教学、存储管理等各项功能，获得一致且高效的使用体验。

---

## 二、个人信息

### 2.1 功能说明

点击桌面中心左下角的菜单图标，在弹出菜单中选择「个人信息」，即可进入个人中心。

### 2.2 可执行操作

| 功能 | 说明 |
| --- | --- |
| **查看个人信息** | 查看账号基本信息，包括用户名、邮箱、注册时间等 |
| **修改信息** | 修改部分个人信息字段（如头像、联系方式等） |
| **实名认证** | 完成实名认证流程，解锁部分高级功能权限 |

---

## 三、费用中心

### 3.1 功能说明

费用中心提供用户算力资源消耗的全面视图，帮助用户了解和管理平台资源使用情况。

### 3.2 主要信息

| 指标 | 说明 |
| --- | --- |
| **总余额** | 用户账户当前拥有的余额总额 |
| **应用消耗总额** | 当前所有运行中应用的余额消耗总量 |
| **消费明细** | 按时间维度展示余额的扣减记录 |

> 余额的扣减规则请参阅 [计费规则](./appendix.md)。

### 4.1 功能说明

平台提供个性化的桌面设置功能，用户可根据自身偏好对桌面界面进行定制化配置。

### 4.2 设置选项

| 设置项 | 说明 |
| --- | --- |
| **主题设置** | 切换桌面的界面主题风格 |
| **操作模式** | 调整桌面的交互操作模式，适配不同用户的使用习惯 |
| **其他个性化选项** | 根据平台提供的可选配置项进行自定义调整 |

## 五、云端互联

### 5.1 功能说明

云端互联是 Hi168 平台的特色功能，通过建立 VPN 虚拟通道，实现云端虚拟机与用户本地环境的网络打通。用户可在本地通过 SSH 等工具直接连接云端虚拟机，无需在本地安装虚拟机。

### 5.2 核心特性

| 特性 | 说明 |
| --- | --- |
| **本地与云端无缝集成** | 本地环境可直接访问云端虚拟机的网络服务 |
| **网络连通** | 本地可直接 ping 通云端虚拟机的 IP 地址 |
| **免本地安装** | 无需在本地安装和配置虚拟机环境，节省本地资源 |
| **低配高用** | 低配置的本地设备即可使用云端高性能集群资源 |

### 5.3 典型使用场景

1. 在本地使用 SSH 客户端（如 PuTTY、Terminal）连接云端虚拟机。
2. 通过本地 IDE（如 VS Code Remote SSH）直接在云端虚拟机上进行开发。
3. 本地浏览器访问云端虚拟机上运行的 Web 服务。
4. 使用本地数据库管理工具连接云端数据库服务。

## 六、帮助

### 6.1 功能说明

平台内置帮助功能，点击后可打开在线帮助文档。文档中详细介绍了平台各个功能模块的使用方法和操作指南，帮助用户快速上手和解决使用过程中遇到的问题。

---

## 七、常见问题

| 问题 | 解决方案 |
| --- | --- |
| 余额不足 | 前往费用中心查看余额可进行充值 |
| 桌面设置未生效 | 刷新桌面页面或重新登录平台 |
| 云端互联连接失败 | 检查 VPN 配置是否正确，确认本地网络可访问平台服务 |
| 个人信息修改失败 | 确认填写的信息格式正确，必填项不可为空 |
