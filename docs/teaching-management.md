# 教学管理

## 概述

Hi168 平台的 **教学管理** 模块提供从课程开发到教务管理的全流程工具链。包括课程的创建、编辑与发布，班级与学员的管理，以及求职就业服务。通过本模块，用户可高效地组织课程内容、管理学习对象，构建完整的在线课程体系。

---

## 一、课程制作

### 1.1 功能说明

课程制作模块提供了一个完整的课程内容管理系统。通过本模块，用户可以创建、编辑和发布高质量的课程，并将视频、实验环境、案例、练习题、课件等多种资源整合到课程体系中。

### 1.2 课程列表

进入「课程制作」后，系统展示当前用户创建的所有课程。课程状态包括：

| 状态 | 说明 |
| --- | --- |
| **制作中** | 课程正在开发编辑阶段，尚未对外发布 |
| **已发布** | 课程已发布到「课程学习」模块，所有用户可浏览和加入 |
| **已归档** | 课程已归档，不再对外展示 |

### 1.3 新建课程

在课程列表页面点击「新建课程」，即可创建一门新课程。创建时需要填写课程的基础信息，包括课程名称、分类、难度等级等。

### 1.4 编辑课程

课程编辑是课程制作的核心环节，涵盖以下内容维度：

#### 1.4.1 课程基础信息

设置课程的基本属性，包括：

- 课程名称与简介
- 课程分类与难度等级
- 课程封面与描述

#### 1.4.2 课程大纲

构建课程的章节结构，定义课程的整体教学框架：

- 新增、编辑、删除章节
- 调整章节排列顺序
- 设置章节层级关系

#### 1.4.3 知识点与实验手册

为每个章节编写详细的知识点内容和实验操作手册：

- 使用富文本编辑器编写知识点说明
- 编写实验步骤与操作指南
- 配置实验环境参数

#### 1.4.4 课程资料关联与上传

为课程章节关联或上传多种类型的教学资源：

- **应用模板**：关联实验环境的应用模板，学生可一键部署实验环境
- **案例**：添加教学案例素材
- **练习**：配置课堂练习题与测验
- **课件**：上传 PPT、PDF 等演示文稿
- **视频**：关联或上传教学视频

#### 1.4.5 协作开发

课程支持多人协作开发模式：

- 添加课程协作成员
- 设置成员的编辑权限
- 多人协同编辑课程内容

#### 1.4.6 权限管理

对课程的访问和编辑权限进行精细化控制：

- 设置课程可见范围
- 配置不同角色的操作权限

#### 1.4.7 需求管理

管理课程开发过程中的需求与任务：

- 创建课程开发需求
- 跟踪需求完成状态
- 管理开发优先级

### 1.5 课程详情

在课程列表中点击任一课程，可进入课程详情页查看完整信息，包括课程的基础信息、目录结构、关联资源等。

### 1.6 发布课程

课程开发完成后，可将其发布到「课程学习」模块，供所有平台用户浏览和学习。

**发布前建议：**

- 检查课程大纲结构是否完整
- 确认所有章节的教学资源已正确关联
- 验证实验环境的应用模板可正常部署
- 预览课程内容，确保排版和内容无误

## 二、教务管理

### 2.1 功能说明

教务管理模块提供班级与学生的综合管理能力。教师可在此创建和管理班级、维护学生信息，并查看班级关联的课程、直播、实验报告等教学数据。

模块主界面设有两个导航标签：**班级管理** 和 **学生列表**。

### 2.2 班级管理

#### 2.2.1 班级列表

进入「教务管理」后，默认展示所有班级的列表信息。

#### 2.2.2 创建班级

在班级管理页面点击「创建班级」，可新建一个教学班级：

1. 填写班级名称及基本信息。
2. 添加班级学生（可从已有学生列表中选择）。
3. 确认创建。

#### 2.2.3 查看班级

点击班级名称可进入班级详情页，查看以下内容：

- 班级基本信息（名称、创建时间、教师等）
- 班级二维码（供学生扫码加入）
- 学生名单
- 班级关联课程
- 直播实训安排
- 实验报告

#### 2.2.4 编辑班级

在班级详情页中可对班级信息进行修改，包括班级名称、描述、学生成员等。

### 2.3 学生管理

#### 2.3.1 学生列表

在教务管理中切换到「学生列表」标签，展示所有学生信息。

#### 2.3.2 查看学生

点击学生记录可查看学生的详细信息，包括个人信息、所属班级、学习进度等。

#### 2.3.3 编辑学生

对学生的个人信息进行修改和维护。

## 三、求职就业

### 3.1 功能说明

求职就业模块面向毕业生和求职者，提供职位信息与招聘服务。该模块结合人工智能技术与大数据分析，为求职者提供智能化、个性化的就业服务，同时为企业提供高效的人才招聘渠道。

### 3.2 岗位列表

进入「求职就业」后，展示所有岗位信息。岗位按状态分为：

| 状态 | 说明 |
| --- | --- |
| **招聘中** | 岗位正在招聘，求职者可查看并投递 |
| **待开放** | 岗位尚未正式开放 |
| **已关闭** | 岗位招聘已结束 |

### 3.3 发布岗位

职业规划师可根据企业招聘需求发布新岗位：

1. 点击「发布岗位」进入编辑页面。
2. 填写岗位 JD（职位描述），包括：
   - 岗位名称与类别
   - 任职要求与职责描述
   - 薪资范围与工作地点
   - 其他招聘信息
3. 确认发布，岗位将出现在岗位列表中。

## 四、教学管理工作流

以下为教师使用教学管理模块的典型工作流程：

```
新建课程 → 编辑课程内容 → 关联教学资源 → 发布课程
                                                ↓
创建班级 → 添加学生 → 关联课程 → 安排直播 → 跟踪教学效果
```

1. **课程准备阶段**：创建课程、编辑大纲、编写知识点与实验手册、关联教学资源。
2. **课程发布阶段**：完成质量检查后发布课程到学习平台。
3. **教务组织阶段**：创建班级、添加学生、关联课程与直播安排。
4. **实施阶段**：开展直播、跟踪学习进度、查看实验报告。
5. **就业服务阶段**：发布相关岗位信息，协助学员就业。
