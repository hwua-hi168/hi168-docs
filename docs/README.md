# Hi168 云平台 — 帮助文档

## 平台简介

Hi168 是一款面向教育领域的云桌面平台，集成了应用部署、课程管理、在线直播、实验教学、镜像管理、存储服务等核心功能模块。平台基于云原生架构，支持容器与虚拟机两种应用运行模式，为教育工作者与学习者提供一站式在线教学与实验环境。

## 适用对象

- **学生 / 学习者**：使用应用中心部署实验环境，参加课程学习与直播教学。
- **教师 / 教育工作者**：制作与发布课程，管理班级与学生，创建直播教学。
- **平台管理员**：管理命名空间、镜像、模板等平台资源。

---

## 快速入门

1. 登录 Hi168 平台，进入云桌面中心。
2. 在 **应用中心** 中选择所需的实验应用，一键部署。
3. 通过 **我的应用** 管理已部署的应用实例，查看运行状态。
4. 进入 **课程学习** 浏览并加入感兴趣的课程，开始学习。
5. 如需长期保存应用状态，可通过 **快照** 或 **模板** 功能进行保存与复用。
