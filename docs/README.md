# Hi168 云平台 — 帮助文档

## 平台简介

Hi168 是一站式云计算服务平台，集成了云端应用部署、对象存储（S3）、云端互联、网络仿真、在线编程、AI 助手、镜像管理等功能模块。支持容器与虚拟机两种应用运行模式，为用户提供开箱即用的云端工作环境。

## 适用对象

- **个人用户**：使用应用中心部署云端应用，使用存储管理文件，搭建网络仿真环境。
- **团队用户**：通过模板和镜像快速复用配置，使用云端互联协同工作。
- **管理员**：管理空间、镜像、模板等平台资源，查看数据看板。

---

## 快速入门

1. 登录 Hi168 平台，进入云桌面中心。
2. 在 **应用中心** 中选择所需应用，一键部署。
3. 通过 **我的应用** 管理已部署的应用，查看运行状态。
4. 使用 **存储管理** 上传和管理您的文件。
5. 通过 **云端互联** 将本地电脑与云端打通，随时访问云上资源。
6. 如需保存应用状态，可通过 **快照** 或 **模板** 功能进行保存与复用。
