# AI Playground

## 概述

Hi168 平台的 **AI Playground** 模块提供统一的大语言模型对话、图像生成、视频生成体验。用户可以在同一界面中切换不同供应商的 AI 模型，上传图片或文件作为上下文参考，进行多轮对话、内容创作和代码编写。

---

## 一、功能说明

### 1.1 核心特性

| 特性 | 说明 |
| --- | --- |
| **多模型切换** | 支持 OpenAI、Anthropic、Google Gemini、xAI 等多家供应商的模型，一键切换 |
| **多轮对话** | 自动维护上下文，支持设置上下文轮数 |
| **图像生成** | 选择图像模型（如 DALL-E、gpt-image-1）即可通过文字描述生成图像 |
| **视频生成** | 选择视频模型（如 Sora 2、Sora 2 Pro）即可通过文字描述生成视频 |
| **图片参考** | 上传图片作为视频生成的参考素材，引导生成风格和内容 |
| **文件上传** | 上传文本文件作为对话上下文，模型可基于文件内容回答问题 |
| **对话管理** | 支持多会话管理、历史记录、对话导出 |
| **参数调节** | 可自定义 Temperature、Top-P、最大 Token 数等模型参数 |

## 二、附件上传说明

Playground 支持两种附件上传方式：**图片上传** 和 **文件上传**，单个文件大小限制为 **5MB**。

### 2.1 图片上传

点击输入框左侧的 **图片按钮**（📷）上传图片，或直接 **粘贴剪贴板中的图片**。

**支持的图片格式：**

| 格式 | 扩展名 | 说明 |
| --- | --- | --- |
| JPEG | `.jpg` `.jpeg` | 通用照片格式 |
| PNG | `.png` | 支持透明背景 |
| GIF | `.gif` | 静态或动态图片 |
| WebP | `.webp` | 高压缩率的现代图片格式 |

**图片用途：**
- **对话模型**：图片作为视觉上下文，模型可分析图片内容并回答相关问题
- **视频模型**：图片作为参考素材（`input_reference`），引导 Sora 生成与图片风格一致的视频

### 2.2 文件上传

点击输入框左侧的 **文件按钮**（📎）上传文本文件。文件内容会作为对话上下文发送给模型。

**支持的文件格式：**

| 分类 | 扩展名 | 说明 |
| --- | --- | --- |
| **文本** | `.txt` | 纯文本文件 |
| **文档** | `.md` | Markdown 文档 |
| **数据** | `.json` `.csv` `.xml` `.yaml` `.yml` | 结构化数据文件 |
| **代码** | `.py` `.js` `.ts` `.go` `.java` `.c` `.cpp` `.h` `.html` `.css` `.sql` | 源代码文件 |
| **脚本/配置** | `.sh` `.bat` `.ini` `.cfg` `.conf` `.toml` `.env` `.log` | 脚本和配置文件 |

> **注意**：不支持 PDF、Word（`.docx`）、Excel（`.xlsx`）等二进制文档格式。如需使用这类文件内容，请先将其转换为纯文本或 Markdown 格式后上传。

### 2.3 上传限制

| 限制项 | 值 |
| --- | --- |
| 单个文件大小 | ≤ 5MB |
| 图片上传方式 | 点击上传、拖拽粘贴 |
| 文件上传方式 | 点击上传 |
| 支持多选 | 是 |

## 三、模型分类

### 3.1 对话模型（Chat）

用于多轮对话、问答、代码编写、文本翻译等。支持上传图片和文件作为上下文。

### 3.2 图像模型（Image）

用于根据文字描述生成图片。选择图像类别的模型后，输入描述即可生成。

### 3.3 视频模型（Video）

用于根据文字描述生成视频。可上传图片作为参考素材引导视频风格。

| 参数 | 默认值 | 说明 |
| --- | --- | --- |
| 分辨率 | 1280×720 | 横版高清 |
| 时长 | 8 秒 | 可选 4/8/12 秒 |
| 参考图片 | 可选 | 上传图片引导生成风格 |

---

## 四、API 令牌

使用 Playground 前需要创建 API 令牌（`sk-nxl-` 开头）。进入「API 令牌」页面创建令牌后，在 Playground 参数设置中填入即可。

令牌也可用于通过 OpenAI 兼容 API 从外部应用（如 Cursor、Continue、VS Code Copilot）接入 Hi168 的模型服务。
