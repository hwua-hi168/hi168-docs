# 存储管理

## 概述

Hi168 平台提供基于 S3 协议的对象存储服务，支持数据的持久化存储、分层计费、分享与跨应用挂载。存储桶（Bucket）类似于个人网盘，用于存储和组织用户的数据文件。

存储管理模块包含以下核心功能页面：

| 页面 | 说明 |
| --- | --- |
| **存储桶列表** | 查看和管理所有存储桶，创建、重命名、删除存储桶 |
| **存储桶详情** | 浏览和操作桶内文件，上传、下载、复制、移动、预览、分享文件 |
| **购买存储** | 购买标准存储空间套餐（包月/包年）和流量加油包 |
| **我的分享** | 管理已创建的所有分享链接，编辑有效期、密码、访问限制 |
| **存储任务** | 查看文件复制、移动等异步任务的执行进度 |
| **存储服务信息** | 查看概览、速率与流量、密钥管理 |

## 一、存储桶列表

### 1.1 功能说明

存储桶（Bucket）是平台提供的基于 S3 协议的对象存储容器，用于存储和组织用户的数据文件。存储管理涵盖存储桶的创建、配置、内容操作、维护和监控等完整生命周期管理，确保数据安全、高效、有序地存储和访问。

### 1.2 列表信息

进入「存储管理」后，系统展示当前用户的所有存储桶列表，包括以下信息：

| 字段 | 说明 |
| --- | --- |
| **桶名** | 用户自定义的桶名称 |
| **存储层级** | 标准存储 / 低频存储 / 冷存储 / 归档存储 |
| **已用容量** | 当前存储桶已占用的存储空间大小 |
| **创建时间** | 存储桶的创建时间 |

在存储桶列表页面，还可查看以下概览信息：

- **基本信息**：已用容量、总容量、各层级桶数
- **存储空间组成**：标准层免费配额使用情况，以及含余额折算的可用上限
- **配额与限速**：当前配额信息、用户级限速设置

此外，列表页提供搜索框，可通过关键字快速检索存储桶。

### 1.3 桶管理操作

在存储桶列表中，可通过两种方式对已有存储桶执行管理操作：

**操作列菜单**（点击行末尾「…」按钮）：

| 操作 | 说明 |
| --- | --- |
| **复制 S3 桶名** | 将存储桶的 S3 标识名称复制到剪贴板，用于 API 调用或第三方工具配置 |
| **删除** | 删除指定的存储桶（需先清空桶内内容） |

**右键菜单**（右击存储桶行）：

| 操作 | 说明 |
| --- | --- |
| **打开** | 进入存储桶详情页，浏览桶内文件 |
| **重命名** | 修改存储桶的自定义名称 |
| **创建快捷方式** | 在桌面上创建存储桶快捷方式，方便快速访问 |

> **⚠️ 重要提示**：删除存储桶之前必须先清空桶内的所有内容（文件和文件夹），否则删除操作将失败。删除后无法恢复。

## 二、创建存储桶

### 2.1 创建方式

平台提供多种创建存储桶的方式，适配不同使用习惯：

| 方式 | 操作路径 |
| --- | --- |
| **菜单创建** | 在存储桶列表页面点击「新建存储桶」按钮 |
| **右击创建** | 在存储管理界面中右击空白区域选择创建 |
| **桌面创建** | 在云桌面上直接右击创建存储桶 |

### 2.2 创建流程

1. 点击「新建存储桶」。
2. 输入存储桶名称（3–10 个字符，支持中文、英文和数字，不能包含符号或空格）。
3. 选择存储层级（详见下方「存储层级说明」）。
4. 确认创建。

创建完成后，可为存储桶在桌面上创建快捷方式，方便快速访问。

### 2.3 存储层级说明

平台支持四种存储层级，不同层级适用于不同的数据访问频率和成本需求：

| 层级 | 特点 | 适用场景 | 计费方式 |
| --- | --- | --- | --- |
| **标准存储** | 高速读写，3 副本高可靠 | 日常频繁读写的文件 | 配额内免费，超出按量计费 |
| **低频存储** | EC 纠删码存储，更经济 | 月均访问 1 次的数据 | 始终按量计费，单价更低 |
| **冷存储** | 最低存储 60 天 | 季度访问的数据，适合备份归档 | 始终按量计费，单价更低 |
| **归档存储** | 最低存储 180 天，价格最低 | 长期归档保留数据 | 始终按量计费，单价最低 |

> **⚠️ 最低存储天数说明**：冷存储和归档存储设有最低存储期限。如果对象在最低天数之前被删除，仍按最低天数收取存储费用（补收剩余天数差额）。例如：归档层对象存储 10 天后删除，仍按 180 天计费。

> **提示**：将不常访问的数据迁移到低频 / 冷 / 归档层，可大幅降低存储成本。标准存储配额内免费，非标准层按量计费无需购买套餐，单价更低。

## 三、存储桶详情（文件管理）

点击进入某个存储桶后，可浏览和管理桶内的所有文件与文件夹。存储桶详情页的使用方式类似个人网盘，提供完整的文件管理能力。

### 3.1 文件浏览

- 以列表形式展示当前目录下的文件和文件夹
- 支持按前缀查询文件夹
- 支持搜索文件名关键字
- 当前文件夹文件过多时，仅显示前 10,000 条

### 3.2 文件与文件夹管理

**工具栏操作**：

| 操作 | 说明 |
| --- | --- |
| **上传文件** | 上传单个文件到当前目录，支持大文件自动分段上传 |
| **上传文件夹** | 上传整个文件夹（含子目录结构） |
| **新建文件夹** | 在当前目录下创建新的文件夹 |
| **批量删除** | 勾选多个文件或文件夹后，点击工具栏的批量删除按钮一次性删除 |

**文件行操作**（每行右侧按钮）：

| 操作 | 说明 |
| --- | --- |
| **下载** | 下载单个文件到本地 |
| **删除** | 删除单个文件或文件夹 |

> **文件上传限制**：单次上传文件数量有上限限制，如需上传大量文件，建议打包压缩后上传，或使用 S3 专用上传工具。上传过程中可在「查看上传进度」中实时查看进度。如果文件已存在，系统会自动跳过上传。

> **文件下载提示**：网页端不支持文件夹下载，请使用桌面端程序进行下载。下载过程中请勿重复点击同一文件。

### 3.3 文件右键操作

右键点击文件，可执行以下操作：

| 操作 | 说明 |
| --- | --- |
| **复制** | 将文件复制到当前桶或其他存储桶的指定路径（弹出路径选择器，支持跨桶复制） |
| **重命名** | 对文件或文件夹进行重命名 |
| **分享文件** | 创建分享链接或分享到论坛 |
| **复制下载链接** | 生成文件的直接下载链接，方便分享 |

> **提示**：文件较大时，复制和移动操作会自动开启加速任务，可在「存储任务」列表中查看进度。

### 3.4 在线预览

存储桶支持对常见格式的文件进行在线预览，无需下载到本地即可查看文件内容。不支持预览的文件类型会提示下载后查看。

### 3.5 文件分享

在存储桶详情页中，可对选中的文件进行分享：

**方式一：创建分享链接**

1. 选择需要分享的文件，点击「分享文件」或「创建链接」。
2. 设置分享参数：
   - **有效期**：限定时间（如 7 天、30 天）或永久有效
   - **提取码**：可设置密码保护，留空则无需提取码
   - **访问限制**：可设置最大访问次数（0 表示不限制）
3. 确认创建，系统生成 **预览链接** 和 **直接下载链接**。
4. 复制链接发送给其他用户。
5. 接收用户通过浏览器打开链接后，可在线预览或直接下载文件，也可将文件 **转存到自己的存储桶** 中。

**方式二：分享到论坛（博客）**

1. 选择需要分享的文件，点击「分享文件」。
2. 选择目标论坛板块（需选择至少一个技术分区）。
3. 确认分享，文件将以博客帖子的形式出现在指定板块中。

> **合规提示**：请勿侵犯他人合法权益，严禁传播色情违法内容，否则封号处理。

## 四、我的分享

「我的分享」页面集中管理用户创建的所有分享链接，方便统一查看、编辑和维护。

### 4.1 分享链接列表

进入「我的分享」后，系统展示所有已创建的分享链接，链接状态包括：

| 状态 | 说明 |
| --- | --- |
| **有效** | 链接在有效期内，可正常访问 |
| **即将到期** | 链接即将过期 |
| **已过期** | 链接已超过有效期，无法访问 |
| **已停用** | 链接已被手动停用 |

列表页还会显示有效链接数和已过期链接数的统计信息，并支持按条件筛选。

### 4.2 编辑分享链接

对已创建的分享链接可进行以下编辑操作：

| 操作 | 说明 |
| --- | --- |
| **有效期调整** | 延长或缩短分享链接的有效期，或设为永不过期 |
| **修改密码** | 输入新密码替换旧密码，或移除密码（设为公开） |
| **最大访问次数** | 设置或修改链接的访问次数限制（0 表示不限制） |
| **复制预览链接** | 复制可在浏览器打开和转存的预览链接 |
| **复制直接下载链接** | 复制文件的直接下载链接 |
| **复制密码** | 复制当前分享链接的提取码 |

### 4.3 停用与重新激活

| 操作 | 说明 |
| --- | --- |
| **停用** | 暂时禁用某个分享链接，外部用户将无法访问 |
| **重新激活** | 将已停用或已过期的链接重新激活，有效期从当前时间起重新计算（默认 7 天） |

### 4.4 删除分享链接

不再需要的分享链接可直接删除。

> **⚠️ 注意**：删除后不可恢复，请确认链接不再需要后再执行删除。

## 五、购买存储

当标准存储免费配额不足时，用户可通过「购买存储」功能扩展存储空间。

### 5.1 进入方式

在存储桶列表页面点击「购买存储」或「订单购买」，进入存储空间购买页面。

### 5.2 存储套餐

购买存储页面包含两个标签页：**存储套餐** 和 **流量加油包**。

页面顶部显示当前概况：
- **存储配额**：已用容量 / 总配额
- **本月流量**：公网流出、已购流量包用量

#### 5.2.1 选择存储套餐

1. **选择计费周期**：支持「包月」和「包年」两种模式，包年享有折扣优惠。
2. **选择套餐容量**：可选 100GB、500GB、1TB、2TB、5TB 等不同档位。
3. **设置数量**：可通过数量选择器购买多份同一套餐。
4. **使用优惠券**（可选）：点击「输入优惠券码」手动输入兑换。
5. **选择支付方式**：
   - **支付宝**：扫码完成支付宝支付
   - **微信支付**：扫码完成微信支付
6. **确认下单**：查看订单汇总（套餐名称、容量、有效期、合计金额），点击「立即下单」。
7. **支付完成**：支付成功后，存储配额将在几分钟内生效。

> **提示**：页面支持按 GB/月、TB/月、GB/日、TB/日 切换价格单位，方便对比不同套餐的性价比。

#### 5.2.2 存储套餐参考价格

| 套餐 | 包月价格 | 包年价格 |
| --- | --- | --- |
| **100GB** | ¥12.00/月 | ¥99.00/年 |
| **500GB** | ¥39.00/月 | ¥349.00/年 |
| **1TB** | ¥49.00/月 | ¥499.00/年 |
| **2TB** | ¥89.00/月 | ¥899.00/年 |
| **5TB** | ¥199.00/月 | — |

> **说明**：以上价格仅供参考，实际价格以页面显示为准。包年相比包月享有更大折扣。

### 5.3 流量加油包

流量加油包用于增加本月公网流出配额。超出免费额度的公网流出流量按量计费（¥0.50/GB），购买加油包可享优惠价 ¥0.45/GB，节省 10%。

#### 5.3.1 购买流量包

| 流量包 | 价格 | 单价 |
| --- | --- | --- |
| **10GB** | ¥5 | ¥0.45/GB |
| **50GB** | ¥23 | ¥0.45/GB |
| **100GB** | ¥45 | ¥0.45/GB |
| **500GB** | ¥225 | ¥0.45/GB |
| **1000GB** | ¥450 | ¥0.45/GB |
| **自定义** | 按量计算 | ¥0.45/GB |

选择或自定义流量包后，选择支付方式（支付宝 / 微信支付），点击「立即下单」完成购买。流量包购买后立即生效，有效期至当月月底，用完即止。

> **何时需要购买流量包？** 当您的文件被大量外部用户下载，产生的公网流出流量超过免费配额时，购买流量包比按量付费更划算。

### 5.4 存储层级与计费

| 层级 | 适用场景 | 按量单价 |
| --- | --- | --- |
| **标准存储** | 频繁访问的热数据，读写性能最高 | ¥0.12/GB/月 |
| **低频存储** | 月均访问 1~2 次的数据 | ¥0.08/GB/月 |
| **冷存储** | 季度访问的数据，最低存储 60 天，提前删除按剩余天数补费 | ¥0.04/GB/月 |
| **归档存储** | 长期归档保留数据，最低存储 180 天，提前删除按剩余天数补费 | ¥0.02/GB/月 |

> **费用说明**：购买的存储空间为标准层免费配额，配额内标准存储免费。非标准层（低频/冷/归档）始终按量计费，单价更低——将冷数据迁至低价层级可显著降低存储成本。

### 5.5 我的订阅

在购买存储页面可查看已购买的存储订阅信息，包括有效期和调整明细。

## 六、存储任务

文件的复制、移动等操作在文件较大时会以异步任务方式执行。进入「存储任务」可查看所有任务的执行状态。

| 信息 | 说明 |
| --- | --- |
| **任务类型** | 文件复制任务 / 文件移动任务 |
| **任务状态** | 等待执行 / 已进入队列 / 执行中 / 已完成 |
| **创建时间** | 任务创建的时间 |
| **预计完成时间** | 系统预估的任务完成时间 |

**支持的操作**：

| 操作 | 说明 |
| --- | --- |
| **取消任务** | 取消正在执行或等待中的任务 |
| **删除记录** | 删除已完成的任务记录 |

## 七、存储服务信息

点击存储桶列表页面右上角的设置图标（⚙），可打开「存储服务信息汇总」面板，包含三个标签页：

### 7.1 概览

展示存储服务的基本信息和配额详情：

| 信息 | 说明 |
| --- | --- |
| **OSS 用户名称** | 当前用户的存储服务用户名，可复制 |
| **OSS 服务地址** | S3 兼容的 API 端点地址，可复制 |

**配额与限速**：
- 存储桶数量上限
- 标准层存储配额与已用容量
- 存储空间组成：初始容量、永久增加、有效期增加、已过期
- 有效期内调整明细（订单权益、注册赠送等，含到期时间）
- 永久调整明细（优惠券兑换等）
- 各层级桶数统计

### 7.2 速率与流量

展示当前用户的网络速率限制和流量使用情况：

**速率信息**：

| 信息 | 说明 |
| --- | --- |
| **下载限速** | 服务端最高下载速率限制（如 ≤20 Mbps） |
| **上传限速** | 服务端最高上传速率限制（如 ≤50 Mbps） |
| **速率来源** | 当前速率等级来源（基础速率 / 付费提升等） |

> 以上为服务端最高限速，实际速率受本地网络、并发负载等因素影响。

**流量配额**：
- 展示当前已购买的流量加油包列表，包含每个包的容量、已用量、有效期和进度条
- 显示总配额与已用流量汇总
- 可直接点击「续购流量包」快速跳转购买

**本月流量统计**：

| 流量类型 | 计费方式 |
| --- | --- |
| **公网流出** | ¥0.50/GB（购买流量包可享 ¥0.45/GB） |
| **上传流量** | 免费 |

### 7.3 密钥管理

提供 S3 兼容的访问密钥管理功能，用于第三方 S3 工具（如 S3 Browser、rclone 等）连接平台存储服务。

| 操作 | 说明 |
| --- | --- |
| **申请新密钥** | 生成新的 S3 访问密钥（Access Key / Secret Key） |
| **查看密钥** | 查看已有密钥信息 |
| **复制密钥** | 将密钥信息复制到剪贴板 |
| **删除密钥** | 删除不再使用的密钥 |

> **⚠️ 安全提示**：请注意保护您的账号安全。删除密钥后，使用该密钥的应用将无法访问存储服务，部分应用可能无法启动，需先解除绑定。

## 八、最佳实践

1. **合理选择存储层级**：日常频繁使用的数据放在标准存储；不常访问的备份数据迁移到低频、冷或归档层，降低存储成本。
2. **利用存储桶持久化数据**：将重要实验数据保存到存储桶，避免因应用到期或删除造成数据丢失。
3. **善用文件分享功能**：通过分享链接实现团队间的文件协作；设置密码和访问次数增强安全性；通过「我的分享」统一管理所有链接。
4. **善用桌面快捷方式**：为常用的存储桶创建桌面快捷方式，提升访问效率。
5. **关注存储配额**：定期检查存储空间使用情况，在标准配额不足时及时购买扩展；包年套餐比包月更划算。
6. **合理使用流量包**：如果文件被大量外部用户下载，建议提前购买流量加油包（¥0.45/GB），比按量计费（¥0.50/GB）节省 10%。
7. **保护 S3 密钥安全**：不要将密钥分享给不信任的第三方，不再使用的密钥及时删除。
8. **删除前备份**：在删除存储桶前，确保已对重要数据进行备份。删除存储桶前需先清空桶内所有内容。
9. **关注速率与流量**：通过「存储服务信息」中的「速率与流量」标签页，定期查看本月流量使用情况，避免产生意料之外的流量费用。
