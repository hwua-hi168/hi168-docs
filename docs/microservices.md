# 平台功能总览

## 简介

Hi168 是一站式云计算服务平台，集成了云端应用部署、对象存储（S3）、云端互联、网络仿真、在线编程与 AI 等能力。各功能模块协同工作，为您提供开箱即用的云端工作环境。

本页面帮助您了解平台由哪些功能模块组成，以及每个模块能为您做什么。

---

## 平台功能模块

### 账号与登录

| 功能 | 说明 |
| --- | --- |
| **用户注册与登录** | 支持密码、短信验证码、微信扫码等多种方式注册和登录 |
| **密码找回** | 忘记密码时，可通过手机号或邮箱重置 |
| **登录状态保持** | 登录后在一定时间内无需重复登录 |

### 云端应用

| 功能 | 说明 |
| --- | --- |
| **应用中心** | 浏览和搜索各类云端应用，一键部署到您的云桌面 |
| **我的应用** | 管理您已部署的应用，查看运行状态，创建快照和模板 |
| **空间管理** | 管理您的云端工作空间，隔离不同项目的资源 |

### 云端互联

| 功能 | 说明 |
| --- | --- |
| **Cloud Link** | 将您的本地电脑与云端环境打通，像使用本地网络一样访问云端资源 |
| **文件传输** | 在本地与云端之间快速传输文件，无需额外工具 |
| **远程操作** | 通过云端互联直接访问和管理云上的应用与设备 |

### 对象存储（S3）

| 功能 | 说明 |
| --- | --- |
| **存储桶管理** | 创建和管理存储桶，分类存放您的文件和数据 |
| **文件上传与下载** | 上传、下载和管理您的云端文件，支持批量操作 |
| **文件分享** | 生成分享链接，方便与他人共享文件 |
| **在线预览** | 无需下载，直接在浏览器中预览文档、图片等文件 |

### 网络仿真

| 功能 | 说明 |
| --- | --- |
| **拓扑搭建** | 在可视化画布上拖拽路由器、交换机等网络设备，连线搭建网络拓扑 |
| **设备模拟** | 支持华为、思科等主流厂商的网络设备模拟，像操作真实设备一样使用 |
| **一键启动** | 搭建好网络拓扑后，一键启动所有设备，秒级进入可用环境 |
| **抓包分析** | 捕获网络流量并分析，直观了解数据包的传输过程 |

### 在线编程与 AI

| 功能 | 说明 |
| --- | --- |
| **Notebook** | 在线编程环境，支持 Python 等多种语言，无需安装任何软件 |
| **AI 助手** | 智能对话助手，可以回答问题、辅助编程、提供建议 |
| **数据集与模型** | 管理项目所需的数据文件与 AI 模型 |

### 镜像与模板

| 功能 | 说明 |
| --- | --- |
| **我的模板** | 将配置好的应用保存为模板，下次一键复用 |

| **模板分享** | 将制作好的模板分享给其他用户使用 |

### 内容与协作

### 账户服务

| 功能 | 说明 |
| --- | --- |

## 典型使用场景

Hi168 的各个功能模块是无缝衔接的，下面举几个典型的使用场景：

### 场景一：搭建网络仿真环境

1. 在**应用中心**找到网络相关应用，点击部署
2. 进入**拓扑管理**，拖拽设备搭建网络
3. 一键**启动**所有设备，开始使用
4. 通过**云端互联**从本地电脑直接访问云上网络设备
5. 用**抓包工具**分析网络流量
6. 将拓扑保存为**模板**，方便下次复用

### 场景二：在线编程与数据分析

1. 打开 **Notebook**，创建一个编程环境
2. 从**对象存储**中导入项目数据
3. 编写代码并运行，查看结果
4. 遇到问题时，向 **AI 助手**提问获得帮助
5. 将输出文件保存到**存储桶**中，生成分享链接

### 场景三：部署云端应用

1. 在**应用中心**选择所需的应用，一键部署
2. 通过**我的应用**查看运行状态
3. 使用**云端互联**从本地直接访问云端应用
4. 满意后创建**快照**保存当前状态
5. 制作**模板**分享给团队成员
