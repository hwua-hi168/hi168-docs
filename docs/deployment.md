# 平台更新与维护

## 简介

Hi168 云平台会定期进行功能升级和系统维护，以确保为您提供最佳的使用体验。平台拥有一套专业的运维管理系统，能够快速、安全地完成版本更新。

---

## 一、平台如何保持更新

Hi168 平台会持续改进，新功能和优化会定期发布。平台更新通常包括：

| 更新内容 | 说明 |
| --- | --- |
| **新功能上线** | 添加新的应用类型、存储功能或交互体验 |
| **体验优化** | 改善界面响应速度、操作流畅度等 |
| **问题修复** | 修复用户反馈的各类问题 |
| **安全加固** | 提升平台的安全性，保护您的数据 |

---

## 二、更新对您的影响

| 情况 | 说明 |
| --- | --- |
| **日常更新** | 大多数更新在后台自动完成，您无需做任何操作，也不会中断您的使用 |
| **计划维护** | 偶尔的大版本升级可能需要短暂维护，届时平台会提前通知您 |
| **数据安全** | 您的文件、应用配置、存储数据等在更新过程中会完整保留 |

---

## 三、遇到问题怎么办

如果您在平台更新后遇到任何异常，可以：

1. **刷新页面** — 按 `Ctrl + F5`（或 `Cmd + Shift + R`）强制刷新浏览器
2. **清除缓存** — 清除浏览器缓存后重新登录
3. **提交工单** — 通过**工单中心**提交问题描述，我们的技术团队会尽快处理
4. **联系客服** — 点击页面右下角的**咨询**按钮，在线咨询客服

---

## 四、平台保障

Hi168 为您提供以下平台保障：

| 保障 | 说明 |
| --- | --- |
| **高可用性** | 平台运行在多台服务器上，即使部分服务器出现故障，平台也能正常运行 |
| **数据备份** | 您的数据会定期自动备份，防止意外丢失 |
| **快速恢复** | 如遇异常，平台支持快速回到上一个稳定版本，最大限度减少影响 |
| **监控告警** | 7×24 小时自动监控，异常情况会第一时间被发现和处理 |

---

> **小贴士**：如果平台更新后某些页面显示异常，请先尝试刷新页面或清除浏览器缓存。这通常能解决大多数显示问题。
