# <font color='#4098f6'>Hi168 云平台使用指南</font>

<h2 style="text-align: center;">欢迎使用 Hi168 云平台</h2>

Hi168 是一站式云计算服务平台，为您提供云端应用部署、对象存储（S3）、云端互联、网络仿真、在线编程与 AI 等服务。打开浏览器即可使用，无需安装任何软件。

---

## 🚀 快速开始

| 我想要... | 怎么做 |
|---------|--------|
| 部署一个应用 | 打开 **应用中心**，选择应用 → 部署 |
| 上传文件到云端 | 打开 **存储管理**，新建存储桶 → 上传 |
| 写代码跑程序 | 打开 **Notebook**，创建环境 → 编程 |
| 搭建网络拓扑 | 打开 **拓扑管理**，拖拽设备 → 连线 → 启动 |
| 本地连云端 | 打开 **云端互联**，下载客户端 → 连接 |
| 遇到问题求助 | 打开 **工单中心**，提交工单 |

---

## 📦 平台功能

### 云端应用
在云端运行各种应用，无需本地安装，打开网页就能用。
- **应用中心** — 一键部署云应用
- **应用模板** — 保存配置，快速复用
- **空间管理** — 管理您的云端工作空间

### 云端互联
将本地电脑与云端环境打通，轻松访问云上资源。
- **Cloud Link** — 本地与云端互通
- **文件传输** — 在本地与云端之间传文件
- **远程访问** — 直接访问云端应用和设备

### 对象存储（S3）
文件、数据集、模型，都可以存在云上，随时访问和分享。
- **存储管理** — 上传、下载、分享文件
- **数据集及模型** — 管理 AI 所需数据

### 网络仿真
搭建虚拟网络环境，练习和验证网络配置。
- **拓扑管理** — 可视化搭建网络
- **设备模拟** — 支持华为、思科等设备
- **抓包工具** — 分析网络流量

### AI 与编程
在线编程和 AI 辅助环境。
- **Notebook** — 在线编程环境
- **AI 助手** — 智能对话与辅助

### 镜像管理
镜像是应用的运行环境，制作一次、随处复用。
- **我的镜像** — 管理您创建的镜像
- **镜像中心** — 浏览公共镜像

### 内容与协作
课程、直播、论坛，与他人交流分享。
- **课程** / **直播** / **论坛** / **讨论**

### 了解平台
全面了解 Hi168 能为您做什么。
- **[平台功能总览](docs/microservices)** — 查看所有功能模块介绍
- **[平台更新与维护](docs/deployment)** — 版本更新与常见问题

### 账户与服务
管理您的账号、费用和会员。
- **费用中心** / **会员中心** / **工单中心**
